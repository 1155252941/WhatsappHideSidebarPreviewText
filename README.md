# WhatsApp Hide Sidebar and Preview Text
A script that can be used in Tampermonkey to enable a icon to hide the left chats bar and always hide the conversation perview when using WhatsApp webpage

Features:
* Hide left chats bar
* Hide preview text (will still show the label when new message come in)
https://github.com/user-attachments/assets/ee504290-aa5d-4f9f-8fdc-c03352fae502
* Show preview text when hover for a second
https://github.com/user-attachments/assets/06a37d6e-7efd-4c01-a532-399e3df2d47c

Steps:
1. Download "TamperMonkey" from extension store
2. Create a new script, paste the .txt file
3. Select "TamperMonkey" --> Manage extension --> Enable user script
4. Enable the new script in WhatsApp web
